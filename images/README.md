# Sample OPG images

Add panoramic dental X-ray images here (JPG or PNG) for the **Sample library** on the dashboard.

- Use wide OPG scans (not regular photos)
- Suggested names: `sample_opg_01.jpg`, `sample_opg_02.png`
- Commit these to GitHub so they appear on Hugging Face Spaces too

The app lists every `.jpg` / `.jpeg` / `.png` in this folder automatically.
